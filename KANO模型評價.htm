「IBM SPSS Web 報告」是互動式報告，它可以在瀏覽器中開啟，並包含 IBM SPSS Statistics 產生的圖表、表格及其他輸出。由於此文件是單個檔案，因此它可以置於任何網站上（無需特殊的伺服器或設定）、張貼於共用檔案伺服器上、複製到可攜式檔案媒體，或透過電子郵件配送。「Web 報告」是 HTML5 檔案，它可以在所有最新版本的常用瀏覽器中開啟。

系統已停用「Web 報告」控制項。您使用的瀏覽器已停用 JavaScript，或者系統已停用 JavaScript 作為安全措施。若沒有 JavaScript，報告會包含與完整功能版相同的所有圖表、表格及其他物件，但無法使用互動式特性。請檢查您的瀏覽器設定是否開啟了 JavaScript，或嘗試使用不同的瀏覽器開啟報告。在平板電腦或智慧型手機上，您可能需要安裝 HTML 檢視器應用程式。

編輯工具列

您可以使用「編輯」工具列來修改表格的外觀，包括變更字型屬性、背景顏色及小數點位數。您還可以向整個表格套用表格外觀，並轉置列和欄。您無法變更表格中的資料值，或編輯表格以外的其他物件。
